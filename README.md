# RETAutismAssessment

This repository contains the supplementary codes for the paper titled "Automated Autism Assessment With Multimodal Data and Ensemble Learning: A Scalable and Consistent Robot-Enhanced Therapy Framework". For this research we used the DREAM dataset (https://github.com/dream2020/data).

If you find this work helpful, consider citing the paper (https://doi.org/10.1109/tnsre.2025.3546519).
